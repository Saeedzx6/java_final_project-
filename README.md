# Car Rental Management System (Java)

A professional-grade Java application designed to automate the workflow of a car rental agency. This project demonstrates core **Object-Oriented Programming (OOP)** principles, including encapsulation, inheritance, and polymorphism, to manage a fleet of vehicles and customer bookings efficiently.

## üöÄ Key Features
* **Fleet Management:** Add, update, and track vehicles (Cars, SUVs, Trucks) with real-time availability status.
* **Booking Engine:** Calculate rental costs based on duration and vehicle type.
* **Customer Records:** Maintain a database of customer information and rental history.
* **Automated Invoicing:** Generate clear billing statements upon vehicle return.
* **Input Validation:** Robust error handling for dates, names, and vehicle IDs.

The project follows a modular structure to ensure clean code:

2.  **Service:** Contains the business logic for calculating prices and checking availability.
3.  **App/Main:** The entry point that handles user interaction (Console or GUI).
